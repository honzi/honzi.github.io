Macro Code Notes
AFK Allows you to quickly leave battlegrounds.
Enable Raw Mouse Can fix issues that occur when holding down mouse buttons.
Focus Pet Useful for classes with pets to show you more detailed information about their state.
Get Build Info Displays information about your client version:
  1. Current WoW version.
  2. Current WoW build number.
  3. Current WoW build date.
  4. Current interface number, useful for addon development.
Reload Graphics Reloads your game graphics, which can fix some graphical issues.
Reload UI Reloads your UI and currently loaded addons. Some changes may require restarting the game.
Target Allows you to easily target any visible targetable entity. Just replace X with the first word of the entity's name.
