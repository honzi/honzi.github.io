Honzi/Guides/Prey Demos
Prey (2006):
  • Open the console with "Ctrl + Alt + ~".
  • Some commands can be set to your "preyconfig.cfg" file to remain set to a specific value, but others must be entered after each load.
Useful Commands preyconfig.cfg Notes
clear Clears the console.
com_showfps x seta com_showFPS "0" Sets the display state of the current frames per second:
  • x=0: don't display fps.
  • x=1: display fps.
exit Exits the entire game.
give x Gives you something:
  • x=all: Gives you every weapon available and refills your health and ammo. Some portions of the game may behave strangely because they assume you do not have specific weapons.
  • x=ammo: refills your ammo.
  • x=health: refills your health.
god Makes you immune to all forms of damage.
kill Kills yourself.
noclip Makes your character fly and pass thru all solid objects. Some portions of the game may behave strangely if you pass thru triggers in a different order.
notarget Makes you invisible to enemies.
pm_crouchspeed x Sets your crouching speed to x. Default is 80.
pm_jumpheight x Sets your jump height to x. Default is 64.
pm_runspeed x Sets your running speed to x. Default is 220.
pm_thirdperson x Sets 3rd person state:
  • x=0: 3rd person state.
  • x=1: 1st person state.
pm_walkspeed x Sets your walking speed to x. Default is 180.
r_brightness x Sets the brightness to x. Default is 1.
r_gamma x seta r_gamma "1" Sets the gamma to x. Default is 1.
setplayergravity x y z Changes the player's gravity on each axis.
spawn x Spawns something:
  • x=monster_centurion: doesn't work in the demo.
  • x=monster_fodder
  • x=monster_hound
  • x=monster_hunter
  • x=monster_mutate: doesn't work in the demo.
