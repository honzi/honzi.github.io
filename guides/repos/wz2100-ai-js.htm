Honzi/Guides/Repos/WZ2100-AI.js
AIs:
  • Disabled: an AI that doesn't do anything.
  • Honzibot: attempts to win competitive games.
  • Honzibot-CyborgSwarm: attempts to win competitive games, but with cyborgs.
  • Honzibot-MissileFortress: attempts to research and build endless missile fortresses.
AI Information:
  • Cheats are not used, including limiting knowledge to sensor range.
Installation:
  1. "git clone https://github.com/honzi/WZ2100-AI.js.git --depth 1"
  2. Copy the contents of "multiplay/skirmish/" into the same folder within your Warzone 2100 installation.
  3. Copy the folders of the bots you want to use into the most updated version folder within the "mods" folder.
Observation:
  1. "shift + backspace" to enable debug mode and "ctrl + o" to open the debug menu.
  2. There are multiple ways to observe an AI:
    • Attach an AI script to the player you are currently observing.
    • Change the "Selected Player" to the AI you want to observe.
  3. "ctrl + +" and "ctrl + -" to increase or decrease the game speed for easier observation.
  4. Utilize various cheats to allow easier testing.
